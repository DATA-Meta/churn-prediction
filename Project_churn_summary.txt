# Netflix Churn Prediction - Project 1

## Overview
This is a machine learning project to predict customer churn for a Netflix-like streaming service.

## Dataset
- 25,000 customers
- Features Age, Country, Subscription Type, Watch Time, etc.
- Time Period March 2024 - March 2025
- Target Churn (45% churned, 55% active)

## Model Performance
- Algorithm Gradient Boosting Classifier
- Accuracy 64.46%
- Precision 61.56%
- Recall 56.52%
- F1-Score 58.93%

## Key Findings
1. Subscription Type is the strongest predictor (52.5% importance)
   - Premium 29% churn (best retention)
   - Standard 46% churn
   - Basic 61% churn (highest risk)

2. Watch Time Hours is very important (36.6% importance)
   - Heavy users (600h) 61% churn
   - Light users (100h) 37% churn

## Business Recommendations
1. Upgrade Basic users to Premium (reduce churn by 50%)
2. Monitor heavy watchers with new content
3. Send personalized recommendations to light users
